# node-boop

[![Build Status](https://secure.travis-ci.org/kilianc/node-bump.png)](https://travis-ci.org/kilianc/node-bump)

nodejs version booper

## Install

`npm install -g bump`

# Usage

    usage: bump [version]
    
## Version Options

    $ bump beep # Patch release
    $ bump boop # Minor release
    $ bump bop  # Major release
